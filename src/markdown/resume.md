## 概要

修士の2年間は機械学習を学び、Webシステム開発の現場ではサーバーサイド・フロントエンド合わせて3年の経験があります。  
現在は開発の現場で25人ほどのエンジニア部門のリーダーを務めています。  
また、戦略コンサルタントとして1年の経験があり、適切な社内調整・資料作成ができます。  

## スキル

#### プログラミング言語とフレームワーク
- **Go**  
  GinによるAPIサーバーの新規開発・本番運用の経験があります。経験年数は1年程です。
  個人的に非常に好きな言語です。gRPCや並行処理についてはプライベートで利用した程度です。
- **Python**  
  修士研究で機械学習に用いていました。現在でも簡単なスクリプトやデータ分析は Python で行っています。
  経験年数は7年程です。複数のウェブ・機械学習FWの経験があります。機械学習は少しブランクがあります。
- **Rust**  
  業務での経験はないですが、趣味の開発は主に Rust を使っています。経験年数は1.5年程です。
  個人的に最も好きな言語です。複数のウェブFW、非同期処理、WebAssembly の経験があります。
- **Kotlin**  
  サーバーの新規開発・本番運用の経験があります。経験年数は 1年弱です。
  ロジックは問題なく組めますが、ライブラリなど技術選定には少し自信がありません。

#### クラウド
- **AWS**  
  業務では3年程使用していました。趣味の開発では VPC・Public/Private Subnet・NAT Gateway など基本的な環境を作り
  ECS Fargate などにデプロイしています。業務で Terraform による環境構築の経験もあります。
- **GCP**  
  業務では3ヶ月ほどデータ分析基盤の構築に使用しました。趣味で CloudRun や GKE にサービスをデプロイしたことも
  ありますが、AWS に乗り換えたため最近はあまり使用していません。

#### DB・ツールなど
- **MySQL**  
  本番環境を含め3年程の経験があります。実行計画をもとに基本的なチューニングが行えます。
- **MongoDB**  
  業務での利用はないですが、趣味で半年程の経験があります。簡単な集計などが可能です。
- **Kubernetes**  
  業務での利用はないですが、趣味で2回ほど、サービスの基盤に採用しました。基本的な設定ファイルの記述・運用が可能です。

### （2020年9月〜）調剤薬局向け SaaS サービスの新規開発案件
【職務】プロジェクトリーダー

【フェーズ】基本設計、開発、テスト

【規模】全体で 60〜70 名、サーバーサイドエンジニアが 30〜20 名.

【主な技術セット】
- 言語・FW：Kotlin（1.3.50）、JDK（11）、SpringBoot（2.2.1）、MySQL（5.7）、Vue.js（2.6）、OpenAPI（3.0）
- インフラ・CI/CD：GitHub Actions、AWS ECS・ECR・Aurora・Elasticsearch

【取り組んだ内容】
- プロジェクト参画当初はサーバーサイドエンジニアとして、基本設計と実装を並行して進めました。
  若手が多いチームだったので、テストのしやすさや計算量を意識した実装について一緒に考えるなど、チームに働きかけました。
- プロジェクトリーダーの離脱をきっかけに、10月から私がその役職に着きました。
  5班にまたがる約30名のエンジニアのマネジメントに従事しました。
- 当時、プロジェクトは大幅に遅延し混乱していたため、現状のタスク量と現実的なスケジュールを
  整理・可視化し、弊社役員やクライアントに展開しプロジェクトの正常化を進めました。
  また、マネジメント層を含む体制の見直し、要件定義漏れ事項のフォロー、クライアント
  の期待値調整など、自らの判断で幅広にプロジェクトへ貢献しました。

### ■ 京都大学大学院 非線形動力学研究室 篠本研
### 画像処理ニューラルネットワークと霊長類の視覚野の類似性について
【主な技術セット】Python（2.7、3.5）、TensorFlow（0.8）、CUDA（7.5）、cuDNN（4.0）、Theano（0.8）、GPU（NVIDIA GeForce Titan）

【研究内容】
- 畳み込みニューラルネットワークの低次層が網膜のニューロンのスパイク活動と似た振る舞いを示すのですが、
  高次層は高次視覚野（V4, IT野）と類似性があるのかを調査しました。
- 事前学習済みの VGG-16 を独自データセットで学習させるために Theano, TensorFlow や GPU を使用しました。
- 統計解析の研究室だったためベイズ統計や基本的な機械学習について勉強しました。
